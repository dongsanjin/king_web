# 王者荣耀移动端官网

> 该项目是一个王者荣耀的webappp项目

## 运行项目
--- bash
# 克隆项目到本地
$ git clone https://github.com/dongsanjin/king_web.git

# 安装项目依赖
$ cd king_web
$ npm install

# 启动服务器端服务
$ cd server
$ npm run serve

# 启动web端服务
$ cd ../web
$ npm run serve
```
